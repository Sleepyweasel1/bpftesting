## Agent skills

### Issue tracker

Issues are tracked as local markdown files under `.scratch/` by feature folder. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage roles use the default vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Domain docs use a single-context layout at repo root. See `docs/agents/domain.md`.
